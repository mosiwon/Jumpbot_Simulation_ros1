# Jumpbot_Simulation
Jumpbot_Simulation in Gazebo/RoS1

Env. : Ubuntu 20.04 ,Ros noetic, gazebo 11



in catkin_workspace -->
```ROS
catkin_make install 
```


```ROS
. install/setup.bash 
```

display jumpbot(gazebo) -->
```ROS
roslaunch jumpbot_gazebo gazebo.launch
```

display jumpbot(rviz2) -->
```ROS
roslaunch jumpbot_gazebo display.launch
```
